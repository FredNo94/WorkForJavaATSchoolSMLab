Тест №1
Проверка ввода размера массива (позитивный)
1. Запустить программу
2. Ввести размер массива любое положительное число
Ожидаемый результат:
Программа будет запрашивать поэлементное заполнение массива

Ожидаемый результат:

Тест №4
Проверка ввода значения массива (позитивный)
1. Запустить программу
2. Ввести размер массива любое положительное число
3. Ввести по очереди все элементы массива положительными или отрицательными числами
Ожидаемый результат:
Программа обработала введенные данные и в файле array.txt появились данные
